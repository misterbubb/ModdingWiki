# Frequently Asked Questions

## Does Crawlspace 2 support mods? {#mod-support}

Crawlspace 2 doesn't have official mod support, but the community has created tools like BepInEx to enable modding. All mods are unofficial and created by the community.

## Do I need BepInEx to use mods? {#bepinex-required}

Yes, BepInEx is the mod loader that allows mods to run in Crawlspace 2. You'll need to install it first before any mods will work.

## Do mods work with the Quest version? {#quest-version}

Crawlspace 2 is only available on PC (Steam). There is no Quest version of the game.

## Does (X Mod) require everybody to have it installed? {#require-all-to-install}

Since Crawlspace 2 is a single-player game, you don't need to worry about mod compatibility with other players. However, if you're using a multiplayer mod, mod compatibility will depend on that specific mod's requirements.

## My game crashes after installing mods. What do I do? {#game-crashes}

1. Check the BepInEx log file at `BepInEx/LogOutput.log` for error messages
2. Try removing mods one at a time to identify which one is causing the issue
3. Make sure all your mods are up to date and compatible with your game version
4. Verify that you installed BepInEx correctly

## Can I get banned for using mods? {#ban-risk}

Crawlspace 2 is a single-player game without anti-cheat, so there's no risk of being banned for using mods. Mod at your own discretion and have fun!

## Where can I find help? {#find-help}

Join the [Crawlspace 2 Modding Discord](https://discord.gg/UApCmJx7U7) for help with modding, troubleshooting, and to connect with other modders.

## How do I uninstall mods? {#uninstall-mods}

To completely remove all mods:
1. Delete the `BepInEx` folder from your game directory
2. Verify game files through Steam if needed

To remove specific mods, delete them from the `BepInEx/plugins` folder.
